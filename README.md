# KoshkiKode — Organization Defaults

This is the [KoshkiKode](https://github.com/KoshkiKode) organization's `.github` repository. It contains the public org profile and community health files that apply to every repository in the organization by default.

---

## Contents

| File / Folder | Purpose |
|--------------|---------|
| [`profile/README.md`](profile/README.md) | Public organization profile shown on [github.com/KoshkiKode](https://github.com/KoshkiKode) |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | Default contributing guide for all KoshkiKode repos |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Default code of conduct for all KoshkiKode repos |
| [`SECURITY.md`](SECURITY.md) | Default security policy for all KoshkiKode repos |

Any file placed here automatically applies as the default for repositories in the organization that do not have their own version of that file.
